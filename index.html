SOLITAIRE

Solitaire is any tabletop game which one can play by one self, usually with cards, but also with dominoes. The term "solitaire" is also used for single-player games of concentration and skill using a set layout tiles, pegs or stones. These games include peg solitaire and mahjong solitaire. The game is most often played by one person, but can incorporate others.
